Internal Team Website
Autodesk Related
After getting changed over to the cloud security team I built out a website that would help clear up confusion on cloud related issues within the companies employee base. This site also allowed for our team to reach back out to our users through blogs, news updates, and git repos.
This website acts as a base point for the cloud security team of Autodesk to get information about themselves out to fellow employees. This allows us to ensure that any team running within our cloud is using the proper amazon AMIs for our tools to scan for security reports.
