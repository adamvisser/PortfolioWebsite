Digital Key Management App
Autodesk Related
Built an app to act as an interface for an already built out code signing framework, helping to ensure security, stability, and longevity of the digital signing certificates used by Autodesk developers.
This app helps to ensure the security of digital private key files. I felshed out features to both on-board new users, watch over signing jobs being done on the remote server, and provide an approval system for access to the signing system.
